# Brand Voice & Style Guide — onlineimageshrinker.com

## Brand Identity
- **Tone:** Friendly, clear, no-BS. Explain things simply without talking down.
- **Persona:** A helpful dev friend who knows both design and code.
- **Values:** Privacy-first, free tools, no uploads to servers, client-side processing.

## Writing Rules
1. **Use active voice.** "Upload your image" not "Your image can be uploaded."
2. **Keep sentences short.** Max 25 words per sentence. Aim for 15.
3. **No jargon without explanation.** If you use "Base64" or "EXIF," define it in the same sentence.
4. **Address the reader directly.** Use "you" and "your" — never "the user."
5. **Bold key takeaways.** Readers scan — make the important bits pop.
6. **Use numbered steps for tutorials.** Always use ordered lists for how-to content.
7. **Include a CTA at the end.** Every article should link to a relevant tool with an action verb.

## Formatting Standards
- **Headings:** H2 for sections, H3 for sub-topics. Never skip levels.
- **Tables:** Use markdown tables for comparisons (format vs format, feature vs feature).
- **Code blocks:** Use fenced code with language identifier (\`\`\`css, \`\`\`html).
- **Images:** No placeholder images. Use generate_image tool if needed.
- **Length:** 800–2000 words per article. Enough for depth, not so much it bloats.

## SEO Guidelines
- **Title format:** "[Action Phrase] — Free Online Tool (2026 Guide)" or similar
- **Meta description:** Under 155 chars. Include primary keyword + value prop + "free."
- **H1:** Matches title tag, one per page.
- **Internal links:** 3–6 per article linking to relevant tools and other blog posts.
- **Structured data:** JSON-LD FAQPage schema for FAQ sections.
- **Image alt text:** Descriptive, includes keyword naturally.

## Content Categories
- **Tutorials** — Step-by-step how-to guides (e.g., "How to Make Image Transparent")
- **Guides** — Comprehensive reference (e.g., "WhatsApp Image Size Guide 2026")
- **Tool Spotlights** — Introducing a specific tool with use cases
